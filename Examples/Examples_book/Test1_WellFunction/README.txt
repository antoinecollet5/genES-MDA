Estimation of hydraulic properties of an aquifer using pumping test data


Problem Description
-------------------
This test demonstrates how to set up genES-MDA to solve a simple analytical case that aims to estimate the transmissivity (T) and storativity (S) of an aquifer.


Dependencies
------------
All required packages and their versions are listed in requirements.txt.

To install the required packages, run:
pip install -r requirements.txt


Execution
---------
To execute the application example, the Python command ES_MDA should be run using the default settings.
